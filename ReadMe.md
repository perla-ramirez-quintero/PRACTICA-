# 🚀 Práctica Mayo - Actividades Educativas Multigrado

Este repositorio contiene los materiales didácticos y digitales para una práctica educativa en un aula multigrado de preescolar (1º, 2º y 3º grado), enfocada en el campo formativo de **Saberes y Pensamiento Científico** de la Nueva Escuela Mexicana (NEM).

## 📂 Contenido de la carpeta "ACTIVIDADES PLANEACIÓN"

- **`exploradores_numericos.html`**: Un juego interactivo (Web App) diseñado para jugar en familia durante el fin de semana. Cubre conteo, comparación de colecciones y números ordinales. Está optimizado para móviles (mobile-first) y no requiere conexión a internet una vez descargado.
- **`planeación_didactica.md`**: El diseño pedagógico completo de la actividad, estructurado bajo los lineamientos de la Nueva Escuela Mexicana.
- **`instrucciones_para_padres.md`**: Una guía redactada de manera cálida para fomentar el involucramiento familiar durante el uso del juego en casa.
- **`toDo.md`**: Lista de verificación personal para asegurar la correcta implementación y seguimiento de la actividad en el aula.

## 🎯 Objetivo Pedagógico
Favorecer el desarrollo de los saberes numéricos como herramienta para resolver situaciones del entorno en diversos contextos socioculturales, adaptando el nivel de complejidad a las edades de los alumnos (3 a 5 años).
